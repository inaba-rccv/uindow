# uindow

Vue3 component library

## Usage

```bash
pnpm add uindow
```

```js
// main.js
import uindow from 'uindow'
import { createApp } from 'vue'
import App from './App.vue'

const app = createApp(App)

app.use(uindow)
app.mount('#app')
```
